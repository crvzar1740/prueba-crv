# 🧠 RAN — Rational Adaptive Network

Bienvenido a **RAN**, un espacio donde exploramos cómo la fatiga mental impacta en tu rendimiento y, sobre todo, cómo evitar que eso juegue en tu contra.

Tomás decisiones todo el día:

- Dinero  
- Salud

👉 Fatiga mental = Peores decisiones
